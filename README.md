Проект по разработке и созданию сайта для банка.  
Является курсовой работой в колледже.  
Для работы проекта необходимо:
## 1. Установка Node.js (желательно последней версии [https://nodejs.org/en/download])
## 2. Создание прототипа сайта через npx create-react-app (иначе, не установятся зависимости, необходимые нам:( )
## 2. Клонирование репозитория
## 4. Установка react-router npm i react-router
## 5. Запуск проекта через npm run
